Dán URL Apps Script Web App vào đây

Cách lấy URL:
1. Mở Google Sheets → Extensions → Apps Script
2. Dán nội dung file apps_script.gs vào
3. Deploy → New deployment → Web app
4. Execute as: Me · Who has access: Anyone
5. Copy URL và dán vào đây
